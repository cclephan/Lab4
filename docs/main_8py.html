Detailed Description

Main file contains interrupt callback function at frequency specified by timer instantiated.

Pins are set up for input/output of RC circuit and a queue holds ADC readings everytime the interrupt callback function is called. Once the queue is full the timer is deinitialized and the entire queue is printed.
